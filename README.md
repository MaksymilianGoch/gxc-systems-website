# Website Starter Kit

Solo-Template für Next.js Marketing-Websites mit Claude Code.

## Stack

- Next.js 15 (App Router)
- TypeScript strict
- Tailwind CSS v4
- Vercel Deployment

## Setup

```bash
# Next.js initialisieren
npx create-next-app@latest . --typescript --tailwind --app \
  --import-alias "@/*" --use-pnpm

# Dependencies
pnpm install

# Dev-Server
pnpm dev
```

## Files

- `CLAUDE.md` — Globale Regeln für Claude Code
- `docs/STACK.md` — Tech-Defaults
